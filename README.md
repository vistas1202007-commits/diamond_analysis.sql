# diamond_analysis.sql
Data analysis of diamond prices using MySQL

Analyzed 50,000+ diamond records to find key pricing trends using MySQL.

## Queries Performed
- Data cleaning — checked missing values and invalid entries
- Average diamond price across the dataset
- Most expensive diamond
- Average price by cut (Ideal, Premium, Good, etc.)
- Most common diamond colors
- Carat vs Price relationship
- Price-per-carat comparison by cut

## Key Findings
- Ideal cut has highest volume but Premium cut has higher avg price
- Color G is most common in dataset
- Higher carat = significantly higher price
- Premium cut gives best value per carat

## Tech Used
- MySQL
- SQL (GROUP BY, ORDER BY, AVG, COUNT, DESC)
